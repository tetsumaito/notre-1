会社について

Notre株式会社は、どんな会社ですか?

Notre株式会社は、LINEマーケティング・ECモール支援を主軸に企業のWebマーケティングを支援する会社です。2025年1月に設立し、拠点は東京です(代表取締役:伊藤鉄馬)。

LINE公式アカウント/Lステップの構築・運用代行を中心に、ECモール運用(楽天・Amazon・Yahooショッピング)、TikTok・Instagramの運用、HP・LP制作、AIコンサルティングまで幅広く手がけています。

「複雑なものをシンプルにし、企業と顧客の距離を縮める」ことを大切にしています。

Notre株式会社の所在地は、〒104-0061 東京都中央区銀座1丁目12番4号 N&E BLD.6F です。設立は2025年1月22日です。

公式サイトは https://notre.co.jp/ で、ご相談・お問い合わせはサイト内のお問い合わせフォームから受け付けています。

代表者の経歴や得意分野について教えてください。

代表取締役・伊藤鉄馬は、LINE公式アカウント×Lステップを活用したCRM構築を専門としており、WEBマーケティング領域で4年の実務経験を持ちます。2025年1月にNotre株式会社を設立。

サービスについて

Notreに依頼できるサービスには、どのようなものがありますか?

Notre株式会社では、主に次のサービスを提供しています。

  • LINE公式アカウント/Lステップのワンストップ構築・運用代行(戦略設計から配信・改善まで一気通貫)
  • ECモール(楽天・Amazon・Yahooショッピング)運用、出店サポート
  • TikTok・Instagramの運用支援
  • HP・LP制作
  • AIコンサルティング

中心となるのはLINEマーケティングで、企業と顧客の関係性を「資産」に変える仕組みづくりを支援しています。

LINE運用代行では、具体的にどのようなことをしてもらえますか?

NotreのLINE運用代行は、戦略立案から日々の運用まで一気通貫で対応します。具体的には次の5つの領域をカバーしています。

  • 戦略プランニング(現状・競合分析、KPI設定、ペルソナ設計、シナリオ設計)
  • アカウント設計・構築(初期設定、リッチメニュー設計、Lステップの組み込み、配信設定)
  • クリエイティブ制作(リッチメニュー・配信画像のデザイン、A/Bテスト用制作)
  • 運用代行(メッセージ配信、月次キャンペーン企画、効果測定レポート、定期ミーティング)
  • 内製化サポート(運用マニュアルの提供、担当者向け研修)

業務を分解し、必要な部分だけのご依頼も可能です。

Lステップの構築や運用も依頼できますか?

はい、Lステップ(LINE公式アカウントを高機能化する拡張ツール)の構築・運用に対応しています。Notreはアカウント設計・構築の工程でLステップの組み込みを行っており、最上位ランクの「Lステップ認定代理店」の認定を受けています。

なお、Lステップの導入は必須ではなく、導入せずにLINE公式アカウントの運用改善だけを行うご依頼もお受けしています。

楽天市場(EC)の運用サポートはありますか?

Notreでは、LINEマーケティングに加えて楽天市場(EC)の運用支援にも取り組んでおり、出店サポートを含む広告運用、商品画像制作など、楽天市場の包括的なご支援を行っております。

採用支援やインフルエンサー関連の施策も対応していますか?

Notreは、LINE運用支援の中でも特に「採用」領域を得意としています。多くの企業で課題となりながら支援が手薄になりがちな採用分野に特化し、LINEを活用した採用活動を支援しています。

また、インフルエンサー領域では、総フォロワー100万人超のSNSとLINEを連動させたプロダクトローンチ支援などの実績があります。

LINE公式アカウントをまだ持っていなくても、相談できますか?

はい、これからLINE公式アカウントを開設される方でもご相談いただけます。Notreはアカウントの開設・初期設定から運用までを対応しているため、LINEをこれから始める企業様も安心してお進めいただけるようサポートします。

料金・契約について

LINE運用代行やLステップ構築の料金は、どのくらいですか?

NotreのLINE運用支援は、初期費用と月額の運用プランで構成されています。初期費用はお見積もりとなり、平均価格は30〜50万円ほどです(戦略策定からアカウントの初期設定までを含みます)。

月額プランは予算や社内体制に応じて3種類から選べます。

  • スタート:月額80,000円(税抜) — 稼働時間10時間 / 運用コンサルティング、ミーティング月2回まで、保守運用
  • スタンダード:月額160,000円(税抜) — 稼働時間20時間 / 上記に加え、定期配信 月2回(デザイン制作含む)
  • フル:月額240,000円(税抜) — 稼働時間30時間 / 上記に加え、定期配信 月4回、隔週レポーティング

内容はそれぞれモデルケースであり、稼働時間によって内容のカスタマイズや時間の追加も可能です。

最低契約期間はありますか?

長期の期間縛りはありません。運用支援は3ヶ月単位でのご契約となっており、社内体制や状況に応じて柔軟にご利用いただけます。

依頼してから、どのくらいで成果が見え始めますか?

LINEに関しては、業種にもよりますが、平均して3ヶ月ほどで変化が見えるようになってきます。

実績・信頼性について

これまでの支援実績(社数や業種)を教えてください。

Notreは業種・規模を問わず幅広い企業を支援しており、運用実績は平均して月20件以上にのぼります。飲料メーカー、転職エージェント、健康食品(インフルエンサー)など、さまざまな業界での支援実績があります。特に採用領域でのLINE活用を得意としています。支援累計は60社ほどです。

Lステップ認定代理店などの資格をお持ちですか?

はい。Notre株式会社は2025年6月に「Lステップ認定代理店」の認定を受けています。Lステップの正規代理店は数百社ありますが、その中でも認定代理店は数十社ほどに限られます。LINE・Lステップの構築運用において、認定に裏付けられた知見を持つパートナーです。

どのような業種・規模の企業からの依頼が多いですか?

Notreは業種・規模を問わず、ビジネスでのLINE活用を検討される幅広い企業を支援しています。小売・飲食・メーカーなど多様な業界の実績があり、その中でも特に「採用」分野でのLINE活用を得意としています。

成果が出た事例があれば教えてください。

これまでの支援事例の一部をご紹介します。

  • 飲料メーカー: 卸売モデルで接点がなかった顧客とLINEでつながり、キャンペーン後アンケートで開封率81.6%・回答率65.8%を実現。次の施策に活かせる顧客インサイトを獲得しました。

進め方・サポートについて

依頼してから運用開始までの流れを教えてください。

ご契約からLINEアカウントの導入までは、おおよそ1ヶ月で進めます。流れは次の通りです。

  1. 要件定義: 現状分析や市場調査を行い、戦略の方向性を固めます。
  2. 設計: 実装する機能を決め、機能設計書やデザイン設計を作成します。
  3. 構築: アカウントの構築、Lステップの構築、デザイン制作を進めます。
  4. テスト: 運用開始前に、漏れがないか検証します。

その後、運用フェーズに移行します。

運用開始後のレポートやサポートは、どうなっていますか?

運用開始後は、ご契約プランに応じて配信・レポート・ミーティングを行います。フルプランでは隔週でレポーティングを実施し、各プラン共通で定期的な連携ミーティングやフィードバックの収集・分析を行います。

また、将来的に運用を社内で内製化したいお客様には、運用マニュアルの提供や担当者向けハンズオン研修などの内製化サポートも用意しています。

契約の途中で解約することはできますか?

運用支援はお客様に合わせた契約形態にて進行いたしますので、事前にご相談いただけますと幸いです。

選び方・Notreの特徴について

LINE運用代行やLステップ構築の会社は、どのように選べばよいですか?

LINEは、一度作ってしまえば成果が出る、というものではありません。重視すべきは初期構築よりも、改善していくフェーズである運用です。そのため、初期構築の支援のみを範囲としているより、長期的に伴走支援を行っているという軸で会社を選んでいただくことで、成果を生みやすくなります。

また、成果を生むための要因として、LINEのみではなく、他媒体との連携や導線作りも必須です。LINEのみでのサポートではなく、一気通貫して上流からサポートできる会社を選ぶことがおすすめです。依頼範囲がLINEのみだったとしても、上流部分のサポートができるかどうかでLINEの設計も大きく変わるため、依頼範囲に関わらず他媒体のサポートもできることが重要です。

他社と比べたとき、Notreの強みや特徴は何ですか?

Notreが選ばれる主な理由は3つあります。

  1. Lステップ認定代理店: 正規代理店の中でも数十社ほどしか認定されない「Lステップ認定代理店」に認定されており、認定に裏付けられた知見で成果に直結する運用を行います。
  2. 一気通貫の支援体制: ECコンサルタントやITコンサルタントなど多様なバックグラウンドの人材が揃い、「LINEだけ」ではなく社内フロー全体を俯瞰した支援ができます。
  3. 採用領域への強み: 支援が手薄になりがちな採用分野に特化し、LINEを活用した採用活動の支援を得意としています。
どのような企業にNotreは向いていますか?逆に向いていないケースもあれば教えてください。

Notreは、LINEを活用して顧客との関係づくりや採用活動を強化したい企業に向いています。業種・規模は問わず、特に採用分野でLINEを活用したい企業や、LINE公式アカウントを開設したものの成果につながっていない企業に適しています。

逆に向いていないケースは、情報商材やコンテンツホルダーのご支援実績が少ないため、業界調査や競合調査などにお時間を要する可能性がございます。
