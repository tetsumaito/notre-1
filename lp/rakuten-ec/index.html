Q & A

Q 成果報酬10%の計算方法は？ +
A前年同月比の売上増加分に対して10%です。
例）前年同月売上500万円→今月600万円 = 増加分100万円 → 成果報酬10万円
※売上が下がった場合は成果報酬なし（固定5万円のみ）
Q 契約期間の縛りは？ +
A最低3ヶ月、以降1ヶ月単位で解約可能です。
Q 途中解約の違約金は？ +
A一切ありません（3ヶ月以降）。
Q モニター期間終了後はどうなりますか？ +
A3ヶ月以降は1ヶ月単位で継続・解約をご選択いただけます。継続の場合も、モニター価格（固定5万円＋成果報酬10%）のままご利用いただけます（事例提供を継続していただく場合）。

事例提供を終了される場合は、通常プラン（月25万円〜）へ移行していただく形となります。
Q モニター5社限定の理由は？ +
A楽天専門チームが手厚くサポートするため、月間5社限定で募集しています。枠が埋まり次第、次回募集（未定）まで受付を停止いたします。
Q 広告費は別途かかりますか？ +
Aはい、楽天RPP広告費は別途実費でご負担いただきます。
※運用代行費（固定5万円＋成果報酬）には含まれません
