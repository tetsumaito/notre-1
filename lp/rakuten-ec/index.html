卸専業の農家をEC参入で初年度年商1億円へ
Before：開店後、月商数万円。EC未経験からの参入で大きな成果を実現。

課題：卸依存からの脱却、利益率改善、自社で価格をコントロールできる販路の確立

実施：EC参入の事業提案・3カ年計画策定・EC専用商品設計・価格戦略（粗利逆算型）・ページ設計・運用設計まで伴走

結果：初年度で年商1億円達成 / 卸より高い粗利率を実現 / 自社ブランド確立 / 卸とECの両立モデルを構築

よくあるご質問

Q & A

Q 成果報酬10%の計算方法は？ +
A前月比の売上増加分に対して10%です。
例）前月売上500万円→今月600万円 = 増加分100万円 → 成果報酬10万円
※売上が下がった場合は成果報酬なし（固定5万円のみ）
Q 契約期間の縛りは？ +
A最低3ヶ月、以降1ヶ月単位で解約可能です。
Q 途中解約の違約金は？ +
A一切ありません（3ヶ月以降）。
Q モニター期間終了後はどうなりますか？ +
A3ヶ月以降は1ヶ月単位で継続・解約をご選択いただけます。継続の場合も、モニター価格（固定5万円＋成果報酬10%）のままご利用いただけます（事例提供を継続していただく場合）。

事例提供を終了される場合は、通常プラン（月25万円〜）へ移行していただく形となります。
Q モニター5社限定の理由は？ +
A楽天専門チームが手厚くサポートするため、月間5社限定で募集しています。枠が埋まり次第、次回募集（未定）まで受付を停止いたします。
Q 広告費は別途かかりますか？ +
Aはい、楽天RPP広告費は別途実費でご負担いただきます。
※運用代行費（固定5万円＋成果報酬）には含まれません
